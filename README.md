# pasar-datos-forms-vectores-matrices
Descripcion del proyecto 

Calcular datos con matrices y vectores

Este proyecto es una aplicacion en Java Swing que muestra un menu principal con varias opciones para realizar operaciones matematicas y visualizar diferentes resultados. Esta organizado en multiples formularios, cada uno encargado de una funcion especifica.

El objetivo general es practicar el uso de formularios en Java, paso de datos, validacion de entradas y despliegue de informacion en ventanas separadas.

1. Formulario principal: menu

El formulario menu es la ventana principal de la aplicacion.
Desde aqui el usuario puede seleccionar entre varias acciones como:

Mostrar el mayor de varios numeros

Mostrar el mayor valor ingresado

Mostrar valores menores o mayores

Mostrar promedios

Calcular sumas de diferentes tipos

Ir a ventanas donde se presentan resultados especificos

Cerrar el programa

El menu funciona como centro de navegacion para acceder a todas las demas ventanas.

2. Formularios de operaciones y resultados

El proyecto incluye varios formularios adicionales, cada uno destinado a una tarea concreta:

Mostrarm / Mostrarmayorm / Mostrarmayorv

Formularios que permiten ingresar varios valores y muestran el mayor numero encontrado.

Mostrarv / Mostrarpv / Mostrarmp

Ventanas donde se muestran valores cargados, promedios o datos procesados.

Sumadem / Sumadev

Formularios dedicados a sumar valores.
Uno se enfoca en suma de elementos y otro en suma de valores individuales.

Salida

Formulario final que permite salir del programa o regresar al menu.

3. Libreria de apoyo

El proyecto incluye una clase independiente llamada libreria, ubicada en el paquete libreria.
Esta clase contiene funciones reutilizables para:

Calculos matematicos

Validacion de datos

Manejo de entrada numerica

Los demas formularios usan esta libreria para procesar los valores ingresados por el usuario.

4. Estructura del proyecto

El proyecto esta organizado asi:

formulario/ → Ventanas y formularios principales

libreria/ → Funciones de soporte

ima/ → Posibles imagenes o recursos graficos

.form y .java → Archivos de interfaz y codigo generado por NetBeans

Proposito del proyecto

Este proyecto fue realizado con fines academicos para:

Practicar el uso de Java Swing y multiples formularios

Organizar un menu general que dirige a diferentes acciones

Desarrollar ventanas que realizan operaciones matematicas

Manejar una libreria propia para reutilizar funciones

Comprender como navegar entre ventanas y mostrar resultados

Es un buen ejemplo de programacion orientada a interfaces graficas, estructuracion de modulos y separacion de logica.
